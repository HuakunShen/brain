# tsup

https://tsup.egoist.sh/

> Bundle your TypeScript library with no config, powered by esbuild.

## What can it bundle?

> Anything that's supported by Node.js natively, namely .js, .json, .mjs. And TypeScript .ts, .tsx. CSS support is experimental.
